Baggrundshistorie

Java er et moderne programerings sprog, som blev designet is starten 1990'erne af Sun Microsystems, men er nu ejet af Oracle.

Java er det, som man kalder "Platform Independent", hvilket vil sige at du kun har brug for at skrive et program en gang, for så at kunne køre det på mange andre platforme.


Mere end 3 miliarder enheder køre Java.

Java bruges til at lave apps for Google's Android OS, mange computer programmer, som f.eks media playere, antivirus programmer, web aplikationer, spil og meget mere.
