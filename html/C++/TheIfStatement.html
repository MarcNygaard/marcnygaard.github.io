The if Statement

Decision Making

The if statement bruges til at aktivere noget kode, hvis a condition(en betingelse) er true(sand).


Syntax:

The condition(betingelsen) angiver hvilket udtryk der skal evalueres. Hvis the condition(betingelsen) er true(sand), så bliver the statements(erklæringen) som er mellem the curly brackets(de krøllede parenteser) executed(aktiveret).

If the condition is false, the statements are simply ignored, and the program continues to run after the if statements body.
