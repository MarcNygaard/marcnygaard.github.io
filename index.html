HTML Basics

HTML eller Hyper Text Markup Language bruges til at skabe selve strukturen af en webpage.

Et HTML dokument skal altid starte med <!DOCTYPE html>. Det <!DOCTYPE html> fortæller dokumentet er at der bruge HTML5, som er den nyeste version af HTML. HTML dokumente i sig selv starter med <html> og slutter med </html>. Det som man kan se på dokumenter er det som er under <body></body>.

HTML består af det som man kalder for tags. Tags er f.eks <html></html> eller <body></body>. Der findes rigtig mange forskellige tages, de bruges til forskellige ting, men de har alle den samme funktion, nemlig at skabe overblik i et HTML dokument.

Læg mærke til hvor teksen er struktureret. F.eks under "body" er der "header", "section" og "footer", de er aller placeret et 'tab' længere ude end "body". På den måde bliver dokumentet nemmer at læse.

Du har nok allerede lagt mæreke til hvordan et tag starter med <> og slutter med</>. Det er sådan at HTML kan kende forskel på hvornår et tag start og slutter.

Er det ikke struktureret som det ovenover, så vil det komme til at se sådan her ud.

Det kan være noget svære at overskue og læse, så derfor er det en god ide at trykke 'tab' hver gang man laver et nyt tag inden under et andet tag.

Du kan i navigationsbaren i toppen af siden gå ind under HTML og læse om de mange forskellige tag.

Der vil også være nogle video og HTML basics, som vil gøre dig endnu klogere på hvordan HTML funger og hvad man kan bruge det til.

CSS Basics

En ting er at kunne skrive noget på en webpage med HTML, men siden vil hurtig blive meget kedlig, for siden vil kun består af sort teskt på hvis baggrund.

Til at endre på det kan man bruge CSS eller Cascading Style Sheets. Som navnet også siger, så bruges det til at style siden. CSS bruges til at farve, placere elementer og meget mere.

Den version af CSS, som bruges nu kaldes CSS3. CSS er bygget og på en helt anden måde end HTML. Du specifere et tag i html ved at skrive det som den tag hedder.....
