HTML, CSS og JavaScript

Informationssamling omkring HTML, CSS og Javascript. Du kan finde videoer og læse om de forskellige sprog.

Start her
HTML

HyperText Markup Language (HTML) er et opmærkningssprog, der primært har til formål at få tekst og billeder på en hjemmeside til at virke som links til andre html-sider. Markup betyder at opmærke, og html definerer netop indhold på websider.

Hver HTML-side kan være en hjemmeside, men oftest vil det være flere html-sider, der ligger under ét domæne, der kaldes en hjemmeside. Et bedre navn er derfor websted, website eller site.

HTML-sproget blev opfundet af medarbejdere hos CERN-atomforskningsinstituttet i Geneve i Schweiz. Tim Berners-Lee var hovedmanden. Formålet var at give forskerne på instituttet et simpelt markup-sprog til deres forskningsresultater på internettet.

CSS

Cascading Style Sheets (CSS) er et sprog som bruges til at style HTML dokumenter. HTML er kun strukturen af en website.

Sammen med HTML og JavaScript er CSS et sprog, der bruges til at oprette et simpelt og brugervenlig wabsite, css bruges også til webapplikationer og til mange mobile applikationer.

CSS's primære funktion er at ændre på opjekter som f.eks layout, farver og skrifttyper.

Før hen var CSS en del af HTML, men man ændrede det senere hen til at være et sprog for sig selv. Der er flere grunde til det, men den støret vil formentligt være at man nemmere kun style flere filer, med kun et css dokument.

JavaScript

JavaScript, ofte forkortet som JS, er et dynamisk sprog der bruges til at skabe liv/bevægelse på hjemmesider.

Sammen med HTML og CSS er JavaScript en af de tre kernesprog der bruges på internettet.

Det bruges til at gøre websider interaktive og skabe online-programmer, som f.eks videospil.

De fleste hjemmesider bruger det, og alle moderne webbrowsere understøtter det uden behov for plug-ins.

Sproget bruges til at lave funktion, som bl.a kan bruges til shopping hjemmesider, spil og meget mere.
